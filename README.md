
## NeedPedia 

### Installation

#### Client Installation

```http
cd client
```
```http
npm install
```

#### Server Installation

```http
cd server
```
```http
npm install --force
```

#### Configure .env
###### Clone the .env.example file to .env
```http
cp .env.example .env
```
###### Open the .env file and paste your key to OPENAI_API_KEY=[YOUR_KEY]
